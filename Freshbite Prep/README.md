# Freshbite Prep

A simple static website for a food delivery and meal planning service.

## Files

- `index.html` — Home page with meal preview carousel and login drawer.
- `meals.html` — Meals selection page with images and nutrient details.
- `billing.html` — Billing page with payment details and delivery summary.
- `plans.html` — Subscription plans page.

## Usage

1. Clone or download the repository.
2. Open `index.html` in your browser to view the site.
3. Alternatively, deploy the repository using GitHub Pages.

## Notes

- Login state is stored in the browser using `localStorage` with key `freshbiteLoggedIn`.
- All navigation links are relative and work within the repository.

## Deployment

For GitHub Pages, enable Pages in repository settings and set the source to the repository root.
